# Spotify Clone

A responsive web-based music player that mimics Spotify's interface and functionality.

## Features

- **Music Categories**: 13 different music collections including Arijit Singh, English hits, Party songs, Chill vibes, and more
- **Audio Player**: Full-featured player with play/pause, previous/next, seek bar, and volume control
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Interactive UI**: Hover effects on cards, animated play buttons, and smooth transitions
- **Playlist Management**: Click on any category to load songs into the playlist

## Tech Stack

- **HTML5**: Structure and audio elements
- **CSS3**: Styling with flexbox, grid, and media queries
- **JavaScript**: Music player functionality and dynamic content loading
- **JSON**: Music data storage

## File Structure

```
Spotify-Clone/
├── index.html          # Main HTML file
├── style.css           # Main stylesheet
├── utility.css         # Utility classes
├── script.js           # JavaScript functionality
├── music-data.json     # Music library data
├── favicon.ico         # Site icon
├── img/                # UI icons and images
└── songs/              # Music files organized by category
    ├── Arijit/
    ├── English-Songs/
    ├── Party/
    └── ...
```

## Setup

1. Clone or download the project
2. Ensure all music files are in the `songs/` directory
3. Open `index.html` in a web browser
4. Start playing music!

## Music Categories

- **English Hits**: Popular international songs
- **Party Songs**: High-energy dance tracks
- **Chill Vibes**: Relaxing and lofi music
- **Mood Off**: Emotional and sad songs

- **Devotional**: Spiritual music
- **Punjabi Songs**: Regional tracks
- **And more...**

## Controls

- **Play/Pause**: Click the main play button
- **Previous/Next**: Navigate between songs
- **Seek**: Click on the progress bar to jump to any position
- **Volume**: Adjust using the volume slider or click to mute
- **Category Selection**: Click any music card to load that category

## Browser Compatibility

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
